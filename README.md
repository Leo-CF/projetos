# Projetos

Coleção de projetos Python.

## Projetos

### binario
Conversor de números binários para decimal com interface gráfica (tkinter).

```
python binario/app.py
```

### calculadora
Calculadora funcional com interface gráfica (tkinter). Suporta +, −, ×, ÷, %, ± e entrada pelo teclado.

```
python calculadora/app.py
```

### pomodoro
Relógio Pomodoro com tomate cartoon. Modos Foco (25 min), Pausa Curta (5 min) e Pausa Longa (15 min). Botões para iniciar, pausar, reiniciar e ajustar o tempo.

```
python pomodoro/app.py
```
